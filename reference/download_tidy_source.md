# Download pre-built tidy parquet and store in cache

Download pre-built tidy parquet and store in cache

## Usage

``` r
download_tidy_source(dataset, url, version)
```

## Arguments

- dataset:

  Character, dataset name

- url:

  Character, download URL

- version:

  Character, dataset version (stored in sidecar)

## Value

Invisible path to cached parquet
