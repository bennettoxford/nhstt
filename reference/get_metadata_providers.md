# Get provider organisation metadata from ODS

Get provider organisation metadata from ODS

## Usage

``` r
get_metadata_providers(use_cache = TRUE)
```

## Arguments

- use_cache:

  Logical, whether to use cached data if available. Default TRUE.

## Value

Tibble with provider organisation metadata

## Examples

``` r
if (FALSE) { # \dontrun{
# Get provider metadata
nhstt_providers <- get_metadata_providers()
} # }
```
