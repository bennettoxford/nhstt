# Get tidy data cache directory for a dataset

Get tidy data cache directory for a dataset

## Usage

``` r
get_tidy_cache_dir(dataset, frequency)
```

## Arguments

- dataset:

  Character, specifying dataset name (e.g., "key_measures_annual",
  "activity_performance_monthly")

- frequency:

  Character, specifying report frequency ("annual" or "monthly")

## Value

Character path to tidy cache directory
