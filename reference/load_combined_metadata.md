# Helper to load and combine main and additional metadata sheets

Helper to load and combine main and additional metadata sheets

## Usage

``` r
load_combined_metadata(dataset_base, period, frequency, use_cache)
```

## Arguments

- dataset_base:

  Base name ("metadata_measures" or "metadata_variables")

- period:

  Reporting period

- frequency:

  Frequency ("annual")

- use_cache:

  Whether to use cache

## Value

Combined tibble
